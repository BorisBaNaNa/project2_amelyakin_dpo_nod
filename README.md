# Primitive DB - Простая база данных

Учебный проект по созданию простой базы данных на Python.

## Установка

### Из исходного кода

```bash
# Клонируйте репозиторий
git git@github.com:BorisBaNaNa/project2_amelyakin_dpo_nod.git
cd project2_amelyakin_dpo_nod

# Установите зависимости
make install

# Соберите пакет
make build

# Установите пакет
make package-install
```
